# UrgenceMaroc.com - Guide des fonctionnalités

Ce document présente les fonctionnalités disponibles selon votre profil : visiteur, pharmacien, annonceur ou administrateur.

---

## Pour les visiteurs

### Trouver une pharmacie

Dès l'arrivée sur le site, la liste des pharmacies s'affiche. Plusieurs options de recherche :

- **Recherche par nom** : tapez "Pharmacie du Soleil" dans la barre de recherche
- **Filtre par ville** : cliquez sur le nom d'une ville (Casablanca, Fès, etc.)
- **Pharmacies de garde** : un onglet dédié liste celles ouvertes 24h/24
- **Vue carte** : repérez les pharmacies autour de vous et cliquez pour les détails
- **Proximité de la gare** : filtre disponible pour les voyageurs

### Consulter les détails

Pour chaque pharmacie, vous trouvez :

- Nom et adresse complète
- Numéros de téléphone (cliquez pour appeler)
- Horaires d'ouverture
- Services proposés (parapharmacie, matériel médical, conseil...)
- Type d'établissement (pharmacie générale, dépôt, hospitalière)
- Position exacte sur la carte
- Badge "vérifié" si les informations ont été contrôlées

### Accéder aux urgences

L'onglet "Numéros d'urgence" affiche :

- Numéros nationaux (Police Secours, Pompiers, SAMU)
- Numéros locaux par ville (hôpitaux, cliniques, protection civile)

Chaque numéro est cliquable pour un appel direct. Des conseils d'urgence sont aussi affichés : rester calme, indiquer sa localisation, décrire la situation.

### Contribuer

Vous connaissez une pharmacie non listée ? Des informations sont fausses ? Vous pouvez aider :

- **Soumettre une localisation GPS** : placez le marqueur sur la carte pour indiquer la position exacte
- **Corriger une information** : proposez un nouveau numéro de téléphone, de nouveaux horaires, etc.
- **Suggérer une pharmacie** : décrivez une pharmacie manquante avec les informations disponibles
- **Envoyer une suggestion** : partagez vos idées pour améliorer le site

Toutes les contributions sont vérifiées avant publication. Vous pouvez laisser vos coordonnées (optionnel) pour être contacté.

## Pour les pharmacies

### Être référencé

Votre établissement peut apparaître sur la plateforme avec :

- Nom et coordonnées complètes
- Adresse et quartier
- Horaires précis
- Liste des services proposés
- Position sur la carte interactive
- Badge "vérifié" après validation par l'équipe

### Service de garde

Quand vous êtes de garde, votre pharmacie bénéficie d'une mise en avant :

- Affichage prioritaire dans les résultats
- Icône distinctive sur la carte
- Présence dans l'onglet "Pharmacies de garde"
- Indicateur de fin de garde visible

La garde peut être programmée avec dates de début et de fin.

### Mise à jour des informations

Si quelque chose change (téléphone, horaires, services) :

- Un utilisateur peut soumettre la correction via le formulaire
- Vous pouvez contacter l'équipe directement
- L'administrateur valide et applique la modification

### Types d'établissement

Trois catégories disponibles :

- Pharmacie générale : établissement standard
- Dépôt pharmaceutique : point de distribution
- Pharmacie hospitalière : rattachée à un hôpital

### Catégories d'emplacement

Pour faciliter la recherche selon la localisation :

- Centre-ville
- Proche de la gare
- Proche de l'hôpital
- Proche de l'aéroport
- Centre commercial
- Proche du marché
- Zone résidentielle

## Pour les annonceurs

### Créer une publicité

Le système publicitaire permet d'afficher :

- **Images** : uploadez votre visuel (JPG, PNG, GIF, WebP)
- **Vidéos** : intégrez une vidéo YouTube, Facebook ou autre plateforme

Chaque publicité inclut :

- Un titre accrocheur
- Un texte descriptif
- Un bouton d'action personnalisable ("En savoir plus", "Profiter de l'offre"...)
- Un lien vers votre site ou page de destination

### Configurer l'affichage

Contrôle total sur la diffusion :

- **Dates** : programmez le début et la fin de votre campagne
- **Priorité** : les pubs à haute priorité sont montrées plus souvent
- **Délai "Passer"** : temps avant que l'utilisateur puisse fermer la pub (5 secondes par défaut)
- **Activation** : activez ou désactivez à tout moment

### Suivre les performances

Pour chaque publicité, statistiques disponibles :

- Nombre de vues (combien de fois affichée)
- Nombre de clics (combien de personnes ont cliqué)
- Taux de clic (pourcentage de conversion)

Ces données permettent d'évaluer l'efficacité de vos campagnes.

## Pour les administrateurs

### Tableau de bord

L'interface d'administration donne une vue d'ensemble :

- Nombre total de pharmacies
- Pharmacies de garde actives
- Soumissions en attente de validation
- Graphiques de fréquentation (7 jours, 30 jours)
- Répartition par ville et par type d'établissement
- Top des pharmacies les plus consultées
- Activités récentes

### Gestion des pharmacies

Actions disponibles :

- **Ajouter** : créer une pharmacie avec toutes ses informations
- **Modifier** : mettre à jour les détails existants
- **Supprimer** : retirer une pharmacie obsolète
- **Gérer la garde** : activer/désactiver avec dates de début et fin
- **Valider la position GPS** : confirmer les coordonnées soumises
- **Marquer comme vérifié** : indiquer que les informations ont été contrôlées

### Traitement des contributions

Quatre types de soumissions à traiter :

**Localisations GPS**
1. Comparez sur la carte la position actuelle et celle proposée
2. Approuvez si correct (les coordonnées sont mises à jour)
3. Rejetez sinon

**Corrections d'informations**
1. Comparez la valeur actuelle et la valeur proposée
2. Approuvez pour appliquer la modification
3. Rejetez sinon

**Suggestions**
1. Lisez le message
2. Répondez si nécessaire
3. Archivez une fois traité

**Propositions de pharmacies**
1. Vérifiez que la pharmacie n'existe pas déjà
2. Contrôlez les informations fournies
3. Approuvez pour créer automatiquement la pharmacie
4. Rejetez sinon

### Contacts d'urgence

Gestion complète des numéros :

- Ajout de nouveaux services (hôpitaux, cliniques, protection civile...)
- Modification des numéros existants
- Organisation par ville ou niveau national
- Ordre d'affichage personnalisable
- Activation/désactivation

Types de services : Police, Pompiers, Ambulance/SAMU, Hôpital, Clinique, SOS Médecins, Protection Civile, Autre

### Paramètres du site

Personnalisation de l'apparence et du référencement :

**Général**
- Nom du site
- Email et téléphone de contact

**Apparence**
- Logo (affiché dans l'en-tête)
- Favicon (icône du navigateur)
- Image de partage (réseaux sociaux)

**Référencement (SEO)**
- Titre et description Open Graph
- Description et mots-clés meta
- Auteur
- URL canonique
- Vérification Google
- Données structurées
- Code personnalisé (en-tête/pied de page)
- ID Google Analytics

### Popups d'information

Messages affichés aux visiteurs :

- Titre accrocheur
- Description principale
- Texte d'avertissement (optionnel, encadré jaune)
- Image (optionnel)
- Option "Afficher une seule fois"
- Ordre d'affichage si plusieurs popups

### Configuration publicitaire

Règles globales du système :

**Activation**
- Activer/désactiver toutes les pubs

**Déclencheur**
- Temps : la pub s'affiche après X secondes
- Nombre de pages : après X pages visitées
- Rechargement : quand la page est rafraîchie

**Répétition**
- Répéter après un intervalle de temps
- Intervalle entre deux affichages

**Limites**
- Maximum de pubs par session
- Pause après "Passer" (secondes)
- Pause après clic (secondes)

**Affichage**
- Sur mobile (oui/non)
- Sur desktop (oui/non)

### Journal d'activité

Suivi des actions et erreurs :

- Requêtes HTTP avec code de réponse
- Temps de réponse
- Adresse IP et navigateur
- Actions administrateur
- Filtrage par niveau (info, warning, error, success)
